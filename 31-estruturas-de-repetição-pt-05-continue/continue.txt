31 - Estruturas de Repetição pt 05 - Continue

O continue é o contrário do break, vai ignorar tudo que está abaixo dele e vai retornar ao for 
e iniciar.

Só vai dar continuidade quando a condição for "falsa", se for verdade, o loop volta ao início,
se for falso pula para a próxima linha.

public class Aula06EstruturasDeRepeticao05 {
    //Dado o valor de um carro, descubra em quantas ele pode ser parcelado
    //Condição valorParcela = 1000
    public static void main(String[] args) {
        double valorTotal = 30000;
        for (int parcela = (int) valorTotal; parcela >= 1; parcela--) {
            double valorParcela = valorTotal / parcela;
            if(valorParcela < 1000){
                continue;
            }
            System.out.println("parcela "+ parcela+ " R$ "+valorParcela);
        }

    }
}
